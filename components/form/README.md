# Form

Customize Form Component.

### Usage

```js
import Form from "../components/form";
```

```jsx
<Form />
```

### Properties

| Props       |      Type      | Required | Values | Default | Description       |
| ----------- | :------------: | :------: | :----: | :-----: | ----------------- |
| `className` |    `string`    |    -     |   -    |    -    | Accepts class     |
| `id`        |    `string`    |    -     |   -    |    -    | Accepts id        |
| `style`     | `obj`, `array` |    -     |   -    |    -    | Accepts css style |
